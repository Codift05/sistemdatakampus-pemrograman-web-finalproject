## 📜 Deskripsi Proyek

Di era digital saat ini, perguruan tinggi menghadapi tantangan dalam mengelola data akademik dan administratif secara efisien. Penggunaan sistem manual seringkali menyebabkan keterlambatan, kesalahan data, dan kesulitan dalam pengambilan keputusan. Proyek **Sistem Data Kampus** ini dikembangkan sebagai solusi digital untuk mengatasi masalah tersebut, menyediakan platform terintegrasi untuk pengelolaan data yang cepat, akurat, dan real-time.

Tujuan utama dari aplikasi ini adalah:
* 🚀 **Otomatisasi Proses Akademik**: Mengganti proses manual dengan sistem digital untuk meningkatkan efisiensi.
* 🔗 **Integrasi Data**: Menyatukan berbagai data akademik dan administratif dalam satu platform terpusat.
* 💡 **Dukungan Pengambilan Keputusan**: Menyediakan data yang akurat untuk mendukung kebijakan kampus.
* 🌟 **Peningkatan Layanan**: Memberikan layanan yang lebih cepat dan tepat kepada mahasiswa serta staf.

## ✨ Fitur Unggulan

Berikut adalah beberapa fitur utama yang telah diimplementasikan dalam sistem:

* 🔐 **Autentikasi Pengguna**:
    * Halaman Login aman untuk berbagai peran pengguna (Admin, Pengguna Terdaftar).
* 👤 **Manajemen Profil Pengguna**:
    * Halaman Profil Saya yang informatif, menampilkan data pribadi pengguna seperti Nama, NIM, Program Studi, Email, dll.
* 🛠️ **Admin - Manajemen Data Mahasiswa**:
    * Tampilan daftar data mahasiswa yang komprehensif (Nama, NIM/NIP, Prodi/Jabatan, Email).
    * Kemampuan untuk melakukan CRUD (Create, Read, Update, Delete) pada data mahasiswa.
    * Form Tambah Data Mahasiswa/Pegawai (saat ini untuk demo frontend, data tidak disimpan permanen).
* 📚 **Konten & Navigasi**:
    * Dashboard User dan Dashboard Admin yang intuitif.
    * Halaman Konten Bebas untuk berbagi inspirasi, tips studi, dan pengumuman kampus.
    * Fitur Logout.

## 👤 Peran Pengguna Sistem

Sistem ini dirancang dengan beberapa peran pengguna:

1.  👑 **Admin**:
    * Akses penuh ke semua fitur manajemen data kampus.
    * Bertanggung jawab atas keakuratan dan pembaruan data.
2.  👨‍🎓 **Pengguna Terdaftar**:
    * Akses ke dashboard pribadi, profil, dan konten umum setelah login.
3.  🌐 **Pengunjung**:
    * Akses terbatas ke halaman informasi publik seperti Tentang Kami dan Konten Bebas.

## 🚀 Instalasi dan Menjalankan Proyek

Ikuti langkah-langkah berikut untuk menjalankan proyek ini secara lokal:

1.  **Clone Repositori**:
    ```bash
    git clone [https://github.com/Codift05/sistemdatakampus-pemrograman-web-finalproject.git](https://github.com/Codift05/sistemdatakampus-pemrograman-web-finalproject.git)
    cd sistemdatakampus-pemrograman-web-finalproject
    ```
2.  **Database Setup**:
    * Buat database baru di MySQL.
    * Import file `.sql` (jika disediakan) untuk struktur dan data awal.
    * Konfigurasikan koneksi database pada file `[NAMA_FILE_KONFIGURASI_ANDA]` (misal: `config.php`, `.env`).
3.  **Instalasi Dependencies**:
    * _(Jika menggunakan PHP/Composer)_:
        ```bash
        composer install
        ```
    * _(Jika menggunakan Node.js/npm)_:
        ```bash
        npm install
        ```
4.  **Jalankan Server**:
    * _(Contoh untuk PHP built-in server)_:
        ```bash
        php -S localhost:8000 -t public
        ```
    * _(Contoh untuk Node.js)_:
        ```bash
        npm start
        ```
5.  **Akses Aplikasi**:
    Buka browser Anda dan navigasi ke `http://localhost:[PORT_ANDA]`.
